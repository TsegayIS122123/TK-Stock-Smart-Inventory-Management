# EthioStock |TK-Stock
Smart Inventory & Sales Management System

## Project Description
EthioStock (TK-Stock) is a front-end web application designed to help Ethiopian small and medium businesses manage inventory and sales digitally. The system replaces manual record-keeping with a modern, user-friendly, bilingual solution.

## Features
- User login (simulated)
- Inventory management (add, update, view products)
- Sales processing with automatic stock updates
- Dashboard with real-time statistics
- English / Amharic language support
- Responsive design for mobile and desktop

## Technologies Used
- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript (DOM manipulation, events, LocalStorage)

## How to Run
1. Download or clone the project
2. Open `index.html` in any modern browser
3. No server or database required

## Project Purpose
This project was developed as part of the INSY3081 course to demonstrate understanding of front-end web development concepts including structure, styling, and interactivity.
